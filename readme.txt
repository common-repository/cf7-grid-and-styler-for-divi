`Grid & Styler For Contact Form 7 And Divi` adds grid and styling support for Contact Form 7. The grid builder sets up form elements a grid in rows and columns. The divi module helps style the contact form 7  

== Description ==

[Grid & Styler For Contact Form 7 And Divi](https://wptools.app/wordpress-plugin/grid-styler-for-contact-form-7-and-divi/?utm_source=wp&utm_medium=plugin-page&utm_campaign=cf7-divi&utm_content=description) plugin adds 

* Grid Layout Builder For Contact Form 7 - To create form in grid layout
* Divi Styler Module For Contact Form 7 - Embed form with default styles and add custom styles (premium) the form using divi module


### Grid Layout Builder For Contact Form 7

https://www.youtube.com/watch?v=s_bvRA4v5Ro


Create a grid layout for contact form 7 using rows and columns (1, 1/2, 1/3, 1/4, 2/3, 3/4). These row and columns can be found on the default contact form 7 template editor. 

For details visit [https://wptools.app/how-to/how-to-create-grid-layout-for-contact-form-7/](https://wptools.app/how-to/how-to-create-grid-layout-for-contact-form-7/?utm_source=wp&utm_medium=plugin-page&utm_campaign=cf7-divi&utm_content=description) 


### Divi Styler For Contact Form 7

https://www.youtube.com/watch?v=0Qjkm3uy9SM

Embed and style contact form 7 on a divi page using a divi module. Details on how to style the form can be found at [https://wptools.app/how-to/how-to-style-contact-form-7-in-divi-theme/](https://wptools.app/how-to/how-to-style-contact-form-7-in-divi-theme/?utm_source=wp&utm_medium=plugin-page&utm_campaign=cf7-divi&utm_content=description)

== Installation ==

How to install the `Grid & Styler For Contact Form 7 And Divi` plugin and get it working.

1. Download the plugin and unzip it. You will get a `cf7-grid-and-styler-for-divi` folder
2. Upload `cf7-grid-and-styler-for-divi` folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
